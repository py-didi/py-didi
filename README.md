## 👑 Главный проект / Featured Project: Auto Merger

Если вы хоть раз пытались руками склеить несколько аналитических отчетов, выгрузок по складам или финансовых таблиц Wildberries / Ozon — вы знаете, что это сущий ад, где легко поехать кукухой.

**Auto Merger** решает эту боль раз и навсегда.

### [🚀 Запустить живое демо в браузере (GitHub Pages)](https://py-didi.github.io/auto_merger/)
### [📦 Посмотреть исходный код репозитория](https://github.com/py-didi/auto_merger)

**Key Features / Что умеет утилита:**
* ⚡ **Мгновенное слияние:** Автоматически находит, обрабатывает и сливает разрозненные файлы в единую структурированную базу данных.
* 📊 **Заточено под маркетплейсы:** Идеально справляется с тяжелыми отчетами WB/Ozon, на которые менеджеры обычно тратят целые дни.
* 🛠 **Zero-Setup:** Работает прямо в веб-интерфейсе — не нужно ничего устанавливать на комп, чтобы склеить таблицы здесь и сейчас.
